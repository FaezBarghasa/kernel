# Devices Module

The `devices` module contains the drivers for the hardware devices that are supported by the kernel.

This module contains the following files:

*   `graphical_debug`: This submodule contains the graphical debug console.
*   `serial.rs`: This file contains the code for the serial port.
*   `uart_16550.rs`: This file contains the driver for the 16550 UART.
*   `uart_pl011.rs`: This file contains the driver for the PL011 UART.
